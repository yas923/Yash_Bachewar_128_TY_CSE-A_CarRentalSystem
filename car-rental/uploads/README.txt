Car Images Upload Folder
========================

This folder stores car images uploaded by the admin.

Image Requirements:
- Supported formats: JPG, JPEG, PNG, GIF
- Maximum file size: 10MB
- Recommended dimensions: 800x600 pixels or higher
- Images will be displayed at 200px height (auto-width)

How to Add Images:
1. Login as admin
2. Go to "Manage Cars"
3. Click "Add Car" or "Edit" on existing car
4. Upload image using the file input
5. Images will be automatically saved to this folder

Note: If no image is uploaded, a default car icon will be displayed.
